global_parsing_cb(a, log=None)

Parsing callback triggered after DataFlowGraph parsing which allows modification of the instruction in the context of the overall computation.

This is primarily used to remodel input-outputs as outputs in jointly destructive instruction patterns (See Section 4.4, https://eprint.iacr.org/2022/1303.pdf).

global_fusion_cb(a, log=None)

Fusion callback triggered after DataFlowGraph parsing which allows fusing of the instruction in the context of the overall computation.

This can be used e.g. to detect eor-eor pairs and replace them by eor3.

static build(c: any, src: str, mnemonic: str, **kwargs: list) → slothy.targets.arm_v81m.arch_v81m.Instruction

Attempt to parse a string as an instance of an instruction.

Parameters:

  • c (any) – The target instruction the string should be attempted to be parsed as.

  • src (str) – The string to parse.

  • mnemonic (str) – The mnemonic of instruction c

  • **kwargs –

    Additional arguments to pass to the constructor of c.

Returns:

Upon success, the result of parsing src as an instance of c.

Return type:

Instruction

Raises:

  • Instruction.ParsingException – The str argument cannot be parsed as an instance of c.

  • FatalParsingException – A fatal error during parsing happened that’s likely a bug in the model.
